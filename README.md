# Gerenciador de Biblioteca — Nível Intermediário (Python)

Desenvolva um sistema de gerenciamento de biblioteca executado no terminal.

O projeto deve ser organizado em módulos, utilizar persistência de dados
e permitir manipulação completa dos registros.

## Funcionalidades esperadas

- Cadastro de livros
- Listagem de livros
- Busca por título
- Atualização de dados
- Remoção por identificador
- Persistência em arquivo JSON

## Requisitos

- Organização do código em módulos
- Tratamento básico de erros
- Persistência automática ao sair do programa

## Execução

```bash
python src/main.py
